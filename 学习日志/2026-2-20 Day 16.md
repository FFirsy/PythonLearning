    def say_hello():
        print("Hello!")

    def add_numbers(a,b):
    return a+b

    def run_system():
        total_additions = 0
        while True:
            print("1. Say Hello")
            print("2. Add Two Numbers")
            print("3. Exit")
            print("4. Show total additions performed")

            choice = input("Enter your choice:")
                if choice == "1":
                say_hello()
            elif choice == "2":
                a = int(input("Enter first number: "))
                b = int(input("Enter second number: "))
                print("The sum is:", add_numbers(a,b))
                total_additions += 1
            elif choice == "4":
                print("Total additions performed: ", total_additions)
            elif choice == "3":
                break
            else:
                print("Invalid choice. Please try again.")
为什么要用 while，而不是多次调用函数？
//while 是系统的生命周期控制器
break 在这里扮演什么角色？
给用户选择是否推出的决定权
主循环是不是“系统心跳”？
是的，主循环是“系统心跳”，它会一直运行，直到用户选择退出。
为什么输入检查必须放在主循环里？
因为他们的输入可能是不合法的，如果不在主循环中做检查，程序会崩溃。
这是不是“边界防御”？
是
如果不做输入检查会发生什么？
当输入不合法的内容时，程序会崩溃。
这个计数变量属于哪个层级？
属于主循环的局部变量
为什么不能写在 add_numbers 内部？
因为add_numbers是一个函数，每次调用计数器都会归零
这是不是一种“系统级状态”？
是。
